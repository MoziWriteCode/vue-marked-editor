# vue-marked-editor

[docs](https://blog.yuanxinfeng.xyz)

## Install

```shell
$ npm install --save vue-marked-editor
```
## Quick Start

```javascript
import Vue from "vue";
import vueMarkedEditor from "vue-marked-editor";
import "vue-marked-editor/vue-marked-editor.css";

Vue.use(vueMarkedEditor);
```
